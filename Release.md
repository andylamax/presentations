# 0.1.0-RC1

## Joyfill Kotlin Multiplatform SDK

It's with great pleasure, I announce to you the very first public release of the 
kotlin multiplatform SDK

### Supported Fields
- Text
- TextArea
- Number
- DropDown
- MultiSelect
- Signature
- DateTime
- Image
- Table

### Supported Callbacks
- onFocus
- onBlur
- onChange
- onFieldChange
- onUpload

### Functionality
- External page Navigation
- Internal page Navigation
- Deleted fields
- Readonly fields

### Supported targets
- [x] Android
- [x] Linux (through JVM)
- [x] Windows (through JVM)
- [x] MacOs (through JVM)
- [ ] Browser (WASM)
- [ ] Ios
- [ ] MacOs (Native)
